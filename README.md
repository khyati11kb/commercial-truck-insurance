# TruckShield AI
Commercial Truck Insurance Pricing & Underwriting Platform built using Machine Learning and Agentic AI.

This repository showcases the evolution of **TruckShield AI**, progressing from a traditional machine learning-based insurance pricing engine to an advanced multi-agent AI underwriting platform with autonomous agents, Retrieval-Augmented Generation (RAG), and explainable AI workflows.
