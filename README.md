# TruckShield AI
Commercial Truck Insurance Pricing & Underwriting Platform built using Machine Learning and Agentic AI.
This repository showcases the evolution of the project from a traditional ML-based pricing engine to an advanced multi-agent AI underwriting system.
---

## Project Versions

### Version 1 – ML Pricing Engine
Location: `Version-1-Basic`

**Highlights**
- Data Validation
- Climate Risk Assessment
- Risk Scoring
- Underwriting Engine
- Premium Pricing
- Streamlit Dashboard
